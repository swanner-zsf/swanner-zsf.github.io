mock

生成随机数据，模拟ajax、axios请求

安装

在src下建立mock.js

在main.js 文件中引入我们创建的 mock.js 文件

使用axios请求

语法规范

  • 属性名 和 生成规则 之间用竖线 | 分隔。

  • 生成规则 有 7 种格式：

    ‘name|min-max’: value

    ‘name|count’: value

    ‘name|min-max.dmin-dmax’: value

    ‘name|min-max.dcount’: value

    ‘name|count.dmin-dmax’: value

    ‘name|count.dcount’: value

    ‘name|+step’: value

生成规则和示例
Mock.mock(rurl,rtype,template|function(options))

当拦截到匹配 rurl 和 rtype 的 Ajax 请求时，将根据数据模板 template 生成模拟数据，并作为响应数据返回，或函数 function(options) 将被执行，并把执行结果作为响应数据返回。

Mock.setup(settings)

配置拦截 Ajax 请求时的行为。支持的配置项有：timeout
指定被拦截的 Ajax 请求的响应时间，单位是毫秒。默认值是’10-100’。

Mock.Random

Mock.Random 是一个工具类，用于生成各种随机数据。
Mock.Random 的方法在数据模板中称为『占位符』，书写格式为 @占位符(参数 [, 参数])。

Mock.Random.extend() 扩展
